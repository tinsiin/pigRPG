# 十日能力・四大ステ 成長レンジ想定

**ステータス: 調査整理（コードベースからの逆算）**

既存コードの定数・閾値・非線形関数から「開発時にどのくらいの値域を想定して設計したか」を読み取り、
十日能力と四大ステの**成長レンジの全体像**をまとめたドキュメント。

> 関連: `doc/スキル計算仕様書.md`, `doc/精神分散防御仕様書.md`

## 1. 十日能力（個別値）のレンジ

### 1.1 コード上の根拠

| 根拠 | ファイル・行 | 示唆するレンジ |
|------|------------|--------------|
| 引き締め値: `NightInkKnight / 10` で段階化、段階5以上で80%ボーナス | `Damage.cs:344-353` | **50以上が「強い」ライン** |
| 引き締め確率: `0.31 + NightInkKnight × 0.01` が1.0に到達 | `Damage.cs:411-432` | **69で確率飽和** → 70前後が有効上限 |
| ジーノの実データ: StarTersi = 45, Rain = 3 | `GeinoData.asset` | 序盤～中盤キャラで**高い能力が40〜50台** |
| 成長にハードキャップなし（`+= growthAmount` のみ） | `StatesNumber.cs:747-767` | 理論上は無限だが、成長速度の逓減で実質的に収束 |

### 1.2 個別十日能力の想定レンジ

| フェーズ | 個別値の目安 | 備考 |
|---------|-----------|------|
| 序盤 | 0〜15 | 初期状態。低い能力は1桁 |
| 中盤 | 15〜45 | ジーノの StarTersi=45 がこのあたり |
| 終盤 | 45〜80 | 引き締め段階5+（=50+）が本格機能、確率式が飽和に向かう |
| やりこみ | 80〜120+ | 確率式は69で飽和済み。ここからは「ご褒美ゾーン」 |

### 1.3 ご褒美ゾーンの意味

引き締め確率は69で100%に達するが、十日能力はそこで止まらない。
80以降の成長が「ご褒美」として機能する領域:

- **四大ステへの線形加算**: 十日能力が伸びる限り ATK/DEF/AGI/EYE も伸び続ける（上限なし）
- **引き締め段階のさらなる上昇**: 段階8, 10, 12... と進み、`CalcTransformCountIncrement` の乱数上限が上がる
- **思えの値（ResonanceValue）**: `十日能力合計 × 0.56` で線形に伸びる
- ただし**成長速度は距離減衰**（距離15で0）により自然に鈍化する

## 2. 十日能力（合計値）のレンジ

十日能力は37種存在するが、キャラの精神属性に応じて**4〜5種が主要能力**として伸びやすい。

### 2.1 合計値の概算

| フェーズ | 主要5種の合計 | 全37種の合計 | 備考 |
|---------|-------------|------------|------|
| 序盤 | 20〜60 | 40〜120 | 主要以外は0〜5程度 |
| 中盤 | 80〜150 | 150〜300 | 成長の方向性が見え始める |
| 終盤 | 150〜300 | 300〜600 | 主要能力が50〜80に到達 |
| やりこみ | 300〜500+ | 600〜1000+ | 逓減はするが成長は止まらない |

### 2.2 合計値を使う計算

| 計算 | 式 | 想定レンジ |
|------|---|----------|
| 思えの値 | `全合計 × 0.56 + Baka × 1.3` | 中盤: 84〜168、終盤: 168〜336 |
| 相手との強さ比率 | `相手合計 / 自分合計` | 1.0前後が同格、2.0以上で格上 |

## 3. 四大ステ（合算値）のレンジ

### 3.1 計算式の構造

```
四大ステ = 基礎値(4.0) + Σ(十日能力 × 共通係数) + Σ(十日能力 × 排他係数 × 適応率)
```

### 3.2 共通係数の大きさ比較

各四大ステで**係数が特に大きい十日能力**:

| 四大ステ | 十日能力 | 係数 | 備考 |
|---------|---------|------|------|
| **ATK** | Raincoat | **22.0** | 桁違いに大きい |
| ATK | Enokunagi | 3.0 | |
| ATK | Blades | 1.0 | |
| ATK | BlazingFire | 0.8 | |
| ATK | Baka | **-11.0** | 大幅マイナス |
| **DEF** | NightInkKnight | 1.3 | |
| DEF | FlameBreathingWife | 1.0 | |
| DEF | Raincoat | 1.0 | |
| DEF | JoeTeeth | 0.8 | |
| **AGI** | Baka | **2.0** | AGIでは最大。ATKと真逆の役割 |
| AGI | HeavenAndEndWar | 1.0 | |
| AGI | BlazingFire | 0.9 | |
| **EYE** | WaterThunderNerve | 1.0 | |
| EYE | NightInkKnight | 1.0 | |
| EYE | FaceToHand | 0.8 | |

### 3.3 四大ステの数値レンジ推定

主要十日能力が50程度のとき、DEF の計算例:
```
DEF = 4.0 + (NightInkKnight=50 × 1.3) + (FlameBreathingWife=30 × 1.0) + (Raincoat=20 × 1.0) + ...小さい項
    ≈ 4 + 65 + 30 + 20 + (雑多な項 ≈ 10〜20)
    ≈ 120〜140
```

| 四大ステ | 序盤 | 中盤 | 終盤 | 備考 |
|---------|------|------|------|------|
| ATK | 5〜30 | 30〜100 | 100〜300+ | Raincoat係数22.0のため、Raincoat育成で急伸する |
| DEF | 5〜25 | 25〜80 | 80〜200 | 係数最大1.3で比較的安定した成長 |
| AGI | 5〜20 | 20〜60 | 60〜150 | 係数最大2.0(Baka)だがBakaはATKを-11.0下げる |
| EYE | 5〜20 | 20〜60 | 60〜130 | 係数最大1.0、比較的緩やか |

### 3.4 コード上の閾値との整合

| システム | 閾値 | 想定四大ステ値 |
|---------|------|-------------|
| 慣れ補正 EYE シグモイド | minEYE=30, maxEYE=**134** | EYE 30〜134 が有効レンジ |
| DEF基礎記憶上昇値 第1段階→第2段階 | DEF = **100** | DEF 100が「ギア変更」ポイント |
| DEF基礎記憶減少値 第1段階→第2段階 | DEF = **88** | DEF 88以降は忘却が極めて緩慢に |
| DEF収束（第2段階decay = 0.0027） | DEF **200+** で実質飽和 | ご褒美ゾーン（200以降は微増） |

**重要**: これらの閾値は `b_DEF` / `b_EYE`（排他係数込みの合算値）に対するもの。
DEF=100 は「中盤終盤の境目」、DEF=200 は「やりこみの到達点」と読める。

## 4. パワーレベルによる変動

四大ステはパワーレベルで倍率変動する箇所がある:

| パワーレベル | MentalDEF倍率 | 精神HP最大値 |
|------------|-------------|------------|
| High | DEF × 0.7 × 1.4 = **0.98倍** | `_hp × 1.3 + _maxhp × 0.08` |
| Medium | DEF × 0.7 × 1.0 = **0.70倍** | `_hp` |
| Low | DEF × 0.7 × 0.7 = **0.49倍** | `_hp` |
| VeryLow | DEF × 0.7 × 0.4 = **0.28倍** | `_hp` |

精神DEFは物理DEFの最大98%（High時）、最低28%（VeryLow時）。

## 5. 成長速度と逓減

### 5.1 成長の仕組み

```
成長量 = グラデーション係数 × スキル十日能力値 × 基礎倍率 × 自信ブースト
```

| 要素 | 値 | 備考 |
|------|---|------|
| 基礎倍率（攻撃スキル） | 0.3 | |
| 基礎倍率（非攻撃） | 0.1 | |
| 自信ブースト | 1.0 or 1.3+Baka×0.01 | 格上撃破時 |
| グラデーション係数 | 0〜1.0 | **距離15で0に減衰** |
| 救済下限 | 0.35 | 係数 < 0.3 かつ距離10以内で発動 |

### 5.2 成長の逓減メカニズム

十日能力にハードキャップはないが、**距離減衰**が自然なソフトキャップとして機能する:

1. キャラの十日能力が伸びるほど、スキルの十日能力との「距離」が開く
2. 距離が15に達すると成長係数が0（=そのスキルからの成長がゼロ）
3. 高レベルスキル（無限成長）は `_infiniteSkillTenDaysUnit × 超過レベル` で十日能力も伸びるが、ここも逓減する

→ **実質的に、特定の十日能力が100を超えるあたりから成長が目に見えて鈍化する**と推測される。

| 中盤 | 20〜60 | 条件が揃えば微量の分散 |
| 終盤 | 60〜120 | 本格的に機能する |
