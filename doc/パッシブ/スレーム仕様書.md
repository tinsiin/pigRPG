# スレーム仕様書

「すれ違うように避ける」カウンター型パッシブ。被弾をトリガーに防御・妨害・隠密の連鎖効果を発動する。
Najtsの技術に由来し、TLOA以外の基本的なアクション的必須有効技術として位置づけられる。

関連仕様書:
- `doc/パッシブ仕様書.md` — パッシブシステム全般
- `doc/命中回避とisBad仕様書.md` — 命中回避と良い/悪い判定

---

## §1 概要

### §1.1 コンセプト

自分の回避率を下げてターゲット率を上げ、あえて敵に狙わせる。
被弾した瞬間にダメージを減衰し、同時に敵を妨害しつつ自分は姿を消す ——
「すれ違うように食らい、威力を殺し、気配を消す」特殊技術。

### §1.2 パッシブ一覧

スレーム系統は1つのスキルから始まり、カスケード（連鎖）で最大6つのパッシブが関与する。

| ID | 名称 | 付与先 | トリガー | 実装クラス |
|----|------|--------|---------|-----------|
| 12 | **スレーム** | 自分 | スキル使用 | `Slaim.cs` |
| 17 | **ジーノ版スレーム** | 自分 | スキル使用 | `geino_Slaim.cs` |
| 10 | **ライティスティニ（展開準備）** | 自分 | ジーノ版スレームのOnBeforeDamage | `Raitistini.cs` |
| 11 | **ライティスティニ効果** | 自分 | ID10の割り込みカウンター反応 | `Raitistinian_MetaDodge.cs` |
| 13 | **リーディングステップ** | 味方 | スレームのOnBeforeDamage | `LeadingStep.cs` |
| 14 | **雲隠れ** | 自分 | スレームのOnBeforeDamage | （汎用パッシブ） |
| 16 | **タクトブルフ** | 敵 | スレームのOnBeforeDamage | （汎用パッシブ） |
| 18 | **背面** | 敵 | イースターノジール反応 | （汎用パッシブ） |
| 19 | **ノジーラ** | 自分 | イースターノジール反応 | （汎用パッシブ） |

## §2 カスケードフロー

### §2.1 全体図

```
スキル使用 → スレーム (ID12/17) 付与
                │
                ▼ 被弾時 (OnBeforeDamage)
    ┌───────────┼───────────────────────┐
    ▼           ▼                       ▼
  雲隠れ(14)  リーディングステップ(13)   タクトブルフ(16)
  自分に付与   味方に付与                敵に付与
  2ターン      スレームと同ターン         十日能力依存の発動率失敗
                │
                ▼ スレーム本体はOnDamageで消滅

─── ジーノ版スレーム (ID17) の追加 ───

    パワー≧普通 の場合：
    └─ ライティスティニ展開準備 (ID10) を自分に付与
          │
          ▼ 割り込みカウンター発生時 (OnInterruptCounter)
          ├─ ライティスティニ効果 (ID11) を自分に付与（3ターン前のめり）
          └─ 前のめり交代阻止 (ID8) を3ターン付与
          └─ ID10は消滅

─── イースターノジール（乱れ攻撃反応）───

  敵の乱れ攻撃（基礎山型補正が-15%以下）検知時：
    ├─ 最終ダメージを50%カット
    ├─ 背面 (ID18) を敵に付与（2ターン基本 + 十日能力補正）
    ├─ ノジーラ (ID19) を自分に付与（同ターン数）
    └─ イースターロック発動（再使用不可）
```

### §2.2 トリガー条件の整理

| フェーズ | トリガー | 条件 |
|---------|---------|------|
| スレーム付与 | スキル使用 | — |
| OnBeforeDamage連鎖 | 被弾直前 | スレームが存在 |
| ライティスティニ付与 | ジーノ版OnBeforeDamage | パワー ≧ 普通 |
| ライティスティニ効果発動 | 割り込みカウンター | ID10が存在 |
| イースターノジール | 敵の乱れ攻撃 | ロック未発動 + パワー条件（§4.2参照） |

## §3 各パッシブ詳細

### §3.1 スレーム本体（ID: 12）

**精神属性**: デビル

| プロパティ | 値 |
|-----------|-----|
| IsBad | false |
| 持続ターン | 4（サイレント練度÷11 で加算、最大8ターン） |
| 歩行残存 | なし |
| 消滅条件 | ターン経過 / OnDamage |

**常在効果:**
- 回避率 66% 低下
- ターゲット率 48%

**OnBeforeDamage時の連鎖処理（`Slaim.cs:22-44`）:**
1. 雲隠れ（ID14）を自分に付与（2ターン）
2. 生存味方全員にリーディングステップ（ID13）を付与
   - AGI 4% 上昇（`SetPercentageModifier`）
   - AGI固定値 += 付与者の盥豚÷3 × 対象のサイレント練度（`SetFixedValue`）
3. タクトブルフ（ID16）を攻撃者に付与（`ApplyTaktbruch`）
4. スレーム本体は被弾で消滅

**ダメージ減衰:**
- 0.38倍（十日能力での比較は「スキルの絶対性を担保するため没」）

### §3.2 ジーノ版スレーム（ID: 17）

**精神属性**: キンダー

スレーム（ID12）を継承。基本ターン数が2つ多い。

**追加処理（`geino_Slaim.cs:12-21`）:**
- パワー ≧ 普通 の場合、ライティスティニ（ID10）を自分に付与

**イースターロックの違い（`geino_Slaim.cs:25-31`）:**
- 通常スレーム: ロック確率100%（一度きり）
- ジーノ版: ロック確率65%（**35%の確率でロックされず再発動可能**）

### §3.3 雲隠れ（ID: 14）

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 2 |
| IsBad | false |

**効果:**
- 気配遮断 — 他の味方がいれば自分はターゲットされない
- 攻撃時に敵の回避率が自分の十日能力分低下

### §3.4 リーディングステップ（ID: 13）

| プロパティ | 値 |
|-----------|-----|
| IsBad | false |

**効果（`LeadingStep.cs`）:**
- AGI固定値加算: PassivePower × 1.2（`AGIFixedValueEffect`）
- AGI割合補正: 1.0 + PassivePower × 0.06（`AGIPercentageModifier`）
- スレームからの付与時: AGI 4%上昇 + 盥豚÷3 × サイレント練度

**フレーバー:**
味方の回避率を支援する — スレーム使用者が気配を消すのと引き換えに、
味方の足運びを導く。

### §3.5 タクトブルフ（ID: 16）

**「リズム破断」— 敵のスキル発動率を失敗させるデバフ**

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 4 |
| IsBad | true |
| 消滅条件 | 死亡 |
| 精神属性耐性 | 支柱（頑固で対抗）、デビル（同族の衝突）は適合しない |

**発動率失敗率の計算（`Slaim.cs:49-96`）:**

攻撃者（タクトブルフを受ける側）と術者（スレーム使用者）の十日能力を比較：

**攻撃者側の値:**
- 盥豚 × 2.2
- スマイラー × 1.4
- 顔から手 × 1.3

**術者側の値:**
- スマイラー - max(0, 攻撃者テント空洞×0.3 - 術者冷酷冷静×0.17)
- max(0, 術者冷酷冷静×2.61 - 攻撃者テント空洞×0.35)

**最終計算:**
1. 能力比率 = 攻撃者値 / 術者値
2. サイレント練度比率 = 攻撃者サイレント / 術者サイレント
3. 平均比率 = (能力比率 + サイレント練度比率) / 2
4. 発動率 = clamp(100 - (平均比率×100 - 100), 0, 100)

**パワーレベル補正:**
- パワーが疲弊: 発動率 × 3（効果が弱まる）
- パワーが高い: 発動率 ÷ 1.9（効果が強まる）

### §3.6 ライティスティニ展開準備（ID: 10）

**割り込みカウンター待ち受けパッシブ**

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 7 |
| IsBad | false |
| 精神属性 | 全種別使用可能 |

**OnInterruptCounter発動時（`Raitistini.cs:12-23`）:**
1. ライティスティニ効果（ID11）を自分に付与
2. 前のめり交代阻止（ID8）を3ターン付与
3. 自身は消滅

### §3.7 ライティスティニ効果（ID: 11）

**3ターン前のめり + ダメージ上限 + 防御強化**

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 3 |
| IsBad | false |
| 終了条件 | ターン経過 / 前のめりでなくなること |

**常在効果:**
- 前のめり状態を維持
- 最大HPの2割以上のダメージを受けない
- 回避率 固定値+10、8%上昇
- 防御値加算: (スマイラー + ドクマムシ) ÷ 3（`Raitistinian_MetaDodge.cs:10`）

**設計思想:**
ライティスティニの本懐は「絶対に死なないこと」。
スレームの「避ける」とは異なり、「手で逸らすように威力を殺す」イメージ。
そのため回避よりも防御値の方が実質的に強い。

### §3.8 背面（ID: 18）

**イースターノジール由来の敵デバフ**

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 2（基本）+ 十日能力補正（最大+3、§4.3参照） |
| IsBad | true |
| 消滅条件 | 死亡 / 攻撃後 |

**効果:**
- 付与者からの防御補正 0.7倍（`AddDefensePercentageModifierByAttacker`）
- 回避率低下: 術者のピルマグレイトフル分（固定値）
- 命中率 -20%（全員に対して）

**フレーバー:**
背中を向かされる動き。くるりと後ろを振り向かせられるイメージ。
イースターノジール祭の「背中から押し合う儀式」に由来。

### §3.9 ノジーラ（ID: 19）

**イースターノジール由来の自己バフ**

| プロパティ | 値 |
|-----------|-----|
| 持続ターン | 2（基本）+ 十日能力補正（背面と同じターン数） |
| IsBad | false |
| 消滅条件 | 死亡 / ダメージ / 攻撃後 |

**効果:**
- 対象者限定の回避補正 +20%（`AddEvasionPercentageModifierByAttacker`）
  - イースターノジール発生時の敵に対してのみ

## §4 イースターノジール

### §4.1 トリガー条件

敵の攻撃が**乱れ攻撃**（基礎山型補正のオフセットが-15%以下）になった時、
パーティ内のスレーム保持者が自動的に反応する。

検知場所: `BaseStates.Damage.cs:566-569` → `BattleGroup.PartySlaimsEasterNoshiirEffectOnEnemyDisturbedAttack()`

### §4.2 発動優先順位とパワー条件

| 優先度 | パッシブ | パワー条件 |
|--------|---------|-----------|
| 1（高） | ジーノ版スレーム（ID17） | ≧ 低い（PowerLevel.Low） |
| 2（低） | 通常スレーム（ID12） | ≧ 高い（PowerLevel.High） |

- パーティ内で**1人分だけ**が実行される
- どちらか一方が実行されたらもう片方は免除
- ジーノ版の方がパワー条件が緩く、優先的に選ばれる

### §4.3 背面・ノジーラのターン数補正

`BattleGroup.DecideNoshiirTurnByAbilityComparison()`

**基本値**: 2ターン

**延長計算:**
1. 術者のピルマグレイトフル vs 敵のサイレント練度を比較
2. 凌駕分: (ピルマ - サイレ) / サイレ の整数部分
3. 最大+3ターン延長（両方に適用）

**星テルシ補正:**
- 星テルシ ≧ 26 かつ相手より多い場合、自分側の比較値が1.1倍

**背面のみの短縮:**
- 敵の天と終戦が一定以上: (天終戦 - 20) / 10 ターン減少（最大-2）

### §4.4 イースターロック

一度発動すると `EasterNoshiirLockKey = true` でロックされ、
そのパッシブ存続中は再発動不可。

| バリエーション | ロック確率 |
|--------------|-----------|
| 通常スレーム（ID12） | 100%（確定ロック） |
| ジーノ版スレーム（ID17） | 65%（**35%で解除** → 再発動チャンスあり） |
