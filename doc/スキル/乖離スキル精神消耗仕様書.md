# 乖離スキル精神消耗仕様書

**ステータス: 実装済み**

自身の十日能力と乖離したスキルを戦闘中に**連発**すると、精神HPが加速的に消耗するシステム。
「使いこなせないスキルを振るい続ける精神的負荷」を、風船が膨らんで破裂するように表現する。

> 関連仕様書: `doc/十日能力システム仕様書.md`（§8.3 苦悩システム）, `doc/ゆりかごシステム仕様書.md`, `doc/精神分散防御仕様書.md`
> 関連メモ: `強い力連発に対するアンチ仕組み.md`, `スキル印象構造と十日能力の乖離による苦悩システム.md`, `スキルレベル.md`

## 1. 概要

### コンセプト

「最強のスキルを何回も使う敵。それをデメリットにする。HxHの念能力のような奥深さ」

風船が膨らみすぎると破裂するように、自分の器に合わない力を**振るい続ければ**精神が壊れる。
1回使う分には耐えられる。連発するから壊れる。

### 苦悩システムとの差別化

| | 苦悩システム | 精神消耗（本仕様書） |
|---|---|---|
| **タイミング** | 戦闘後（勝利時） | 戦闘中（リアルタイム） |
| **判定** | 「7回以上使って71%超が乖離スキルだったか」一発判定 | 使うたびに回数で加速する累積判定 |
| **効果** | 十日能力値が減少（長期的弱体化） | 精神HPが減少（即座にダウナー乖離リスク） |
| **「連発」への感度** | 割合で判定。回数自体は問わない | 回数が増えるほど消耗が加速する |

## 2. 乖離判定（苦悩システムと共有）

乖離判定のロジックは苦悩システム（`十日能力システム仕様書.md` §8.3）と**同一の定義**を使用する。
スキルの印象構造全体 vs キャラの十日能力全体を比較する方式。

### 2.1 スキルが「乖離している」とみなす条件

> 原典: `スキル印象構造と十日能力の乖離による苦悩システム.md`

```
① 判定する十日能力の種類数
   judgeCount = max(1, floor(スキルの印象構造の種類数 × 0.8) - 1)

② judgeCount個をランダムに選ぶ

③ 選ばれた各十日能力について:
   デフォルト精神属性互換の全精神属性との距離の平均を計算

④ ③の全平均が 8 以上なら → 乖離スキルとみなす
```

**補足**: 「-1する理由はスキル印象構造の種類が少なくても、8割だけなら全然引かれないから」「1クランプ（最低一つは判定する）」

### 2.2 乖離度（距離の平均値）

乖離判定（§2.1④）で算出される「距離の平均値」を**乖離度**と呼ぶ。
この値は精神HP消耗量の算出（§3）と精神DEF低下の発動判定（§4）の両方で使用する。

- 乖離度 < 8: 乖離スキルではない → 消耗なし
- 乖離度 ≥ 8: 乖離スキル → 精神HP消耗が発動
- 乖離度 ≥ 12: 極端な乖離 → 精神DEF低下も発動

### 2.3 判定の共有について

苦悩システムは「戦闘全体を通じて乖離スキルを何回使ったか」を集計するが、
精神消耗は「今まさに使ったスキルが乖離しているか」を個別に判定する。

判定基準（何をもって乖離とみなすか）は同一。使うタイミングと効果が異なる。

## 3. 精神HP消耗（主効果: 連発で加速）

### 3.1 発動条件

乖離スキル（乖離度 ≥ 8）を**使用した瞬間**に発動する。攻撃が当たったかどうかは問わない。

- 発動判定は攻撃者自身に対して行われる（自傷的な消耗）
- 敵味方共通

### 3.2 連発カウント

戦闘中の乖離スキル使用回数 `N` を追跡する。

- 戦闘開始時に `N = 0`
- 乖離スキルを使用するたびに `N += 1`
- 戦闘終了時にリセット

### 3.3 消耗量の計算

```
使いこなし度 = Σ actor.BaseTenDayValues[スキルの印象構造キー] / skill.TenDayValuesSum
基礎消耗 = max(0, 1 - 使いこなし度 / K) × 係数   ※K = 2.0, 係数 = 0.04

精神HP消耗 = 基礎消耗 × f(N) × MentalMaxHP       ※f(N) = N（線形）
```

#### 使いこなし度と基礎消耗の関係

ゆりかごシステムと**同じ式**（`Σ actor[matching keys] / skill.TenDayValuesSum`）を使うが、向きが逆。
ゆりかごでは使いこなし度が高いほど揺れが大きくなるが、精神消耗では使いこなし度が低いほど消耗が大きくなる。

`K` は「完全に無消耗になるために必要な使いこなし度」の調整ノブ。
スキルが要求する印象構造に対して**十分に大きい器**を持っていなければ、連発の消耗からは逃れられない。

| 使いこなし度 | K=2.0の場合 | 意味 |
|------------|-----------|------|
| 0.2 | 0.9 × 係数 | ほぼ合ってない。重い消耗 |
| 0.5 | 0.75 × 係数 | 半分合ってる。まだ重い |
| 1.0 | **0.5 × 係数** | **釣り合ってても消耗する** |
| 1.5 | 0.25 × 係数 | かなり上回ってる。軽い消耗 |
| 2.0 | 0 | 倍の器がある。やっと無消耗 |

> **「強いスキルを連発するデメリット」の核心**: 使いこなし度1.0（スキル要求とちょうど釣り合ってる）でも
> 基礎消耗は0にならない。器がスキルを**大きく上回って**初めて無消耗になる。
> 強いスキルほど印象構造の総量が大きいため、無消耗ラインに届きにくい。

#### 3システムでの使いこなし度の使い分け

| システム | 式 | 使い方 |
|---------|-----|--------|
| **ゆりかご** | mastery | 高い → 揺れが**大きい**（共鳴する） |
| **苦悩** | 乖離判定（距離ベース） | 乖離 → 戦後に十日能力が**落ちる** |
| **精神消耗** | max(0, 1 - mastery/K) | 低い → 精神HPが**削れる** |

同じ物差しの三面。新しい計算式を発明する必要がない。

#### 連発加速 f(N)

`f(N) = N`（線形）。1回目は軽く、連発するほど線形に重くなる。

```
  消耗
  ↑        ╱
  │       ╱
  │     ╱    ← 連発するほど加速（風船が膨らむ）
  │   ╱
  │  ╱
  │╱
  └──────────→ 使用回数 N
```

5回目は1回目の5倍の消耗。シンプルだが、使いこなし度の基礎消耗との掛け算で十分な抑止力になる。

### 3.4 パラメータ一覧

| パラメータ | 影響 | 値 |
|-----------|------|-----|
| K（無消耗しきい値） | 使いこなし度がK以上で消耗ゼロ。大きいほど厳しい | **2.0** |
| 係数 | mastery=0, N=1 で精神MaxHPの何%を消耗するか | **0.04**（4%） |
| f(N) | 連発加速の曲線 | **N**（線形） |

### 3.5 ゲームプレイ上の意味

- **1〜2回の使用**: 消耗はわずか。状況に応じて乖離スキルを使う判断は有効
- **3〜5回**: 消耗が目立ち始める。連発を続けるかの判断が問われる
- **6回以上**: 消耗が急激。精神HPが危険域に入り、ダウナー乖離が現実的になる
- **単純AIの敵**: 判断せず連発するため、自然に自滅していく
- **器の大きいキャラ**: 十日能力がスキル要求の倍近くあれば消耗なしで連発可能。それが「使いこなしている」ということ

## 4. 精神DEF低下（副次効果: 連発で段階進行）

メモの記述:
> 「精神防御力が半減、4分の一倍まで下がる(特別処理)」

### 4.1 主効果と同じ構造

副次効果は主効果と**同じ計算構造**を持ち、パラメータだけが異なる。
どちらも「使いこなし度ベースの基礎値 × 連発加速」で動く。

```
主効果（§3）:
  ゲート: 乖離度 ≥ 8
  基礎消耗 = max(0, 1 - mastery / 2.0) × 0.04
  精神HP -= 基礎消耗 × N × MentalMaxHP

副次効果（§4）:
  ゲート: 乖離度 ≥ 12
  副次蓄積 += max(0, 1 - mastery / 3.0) × N
  副次蓄積 ≥ 3.0 → 精神DEF × 0.5
  副次蓄積 ≥ 6.0 → 精神DEF × 0.25
```

| | 主効果 | 副次効果 |
|---|---|---|
| **ゲート** | 乖離度 ≥ 8 | 乖離度 ≥ 12（より厳しい） |
| **K値** | K = 2.0 | K_def = 3.0（Kより高い = 打ち消しにくい） |
| **Nの役割** | 消耗量が加速 | 蓄積が進んで段階が進行 |
| **出力** | 精神HP減少（連続値） | 精神DEF低下（段階式: ×0.5 → ×0.25） |
| **f(N)** | N（線形、共通） | N（線形、共通） |

### 4.2 副次蓄積と段階進行

副次蓄積は戦闘中に累積し、閾値を超えるたびに精神DEFの段階が進む。

```
副次蓄積 += max(0, 1 - mastery / 3.0) × N   ※乖離度≥12の使用ごとに加算
```

| 段階 | 精神DEF倍率 | 条件 |
|------|-----------|------|
| 通常 | ×1.0 | 副次蓄積 < 3.0 |
| 第1段階 | ×0.5 | 副次蓄積 ≥ 3.0 |
| 第2段階 | ×0.25 | 副次蓄積 ≥ 6.0 |

- 連発しなければ蓄積が進まず、段階も進まない
- K_def=3.0が主効果のK=2.0より高いため、主効果を打ち消せる器の持ち主でも副次効果は蓄積しうる
- N（線形加速）があるため、連発するほど段階進行が早まる

### 4.3 パラメータ一覧

| パラメータ | 影響 | 値 |
|-----------|------|-----|
| K_def（副次無蓄積しきい値） | 使いこなし度がK_def以上で蓄積ゼロ。Kより高く設定 | **3.0** |
| S1（第1段階閾値） | 副次蓄積がこの値で DEF × 0.5 | **3.0** |
| S2（第2段階閾値） | 副次蓄積がこの値で DEF × 0.25 | **6.0** |
| 高閾値（乖離度ゲート） | この乖離度以上で副次効果が蓄積を開始 | **12** |

### 4.4 ダウナー乖離への波及

精神DEFが低下すると、精神ダメージを受けた際に精神HPがより多く減る。
→ ダウナー乖離（精神HP < 物理HP）の条件を満たしやすくなる。
→ 既存の乖離システム（しきい値判定・パッシブ付与）がそのまま機能する。

これが「標準ロジック ダウナー乖離が起こりやすくなる」の意味。
新しい乖離判定を追加する必要はなく、既存のダウナー乖離が**自然に**誘発される。

副次効果は主効果の**触媒**。主効果だけでもダウナー乖離に至れるが、副次効果が加わると相手の攻撃による精神ダメージも増大し、崩壊が加速する。

## 5. ダウナー乖離との相互作用フロー

```
[乖離スキルを使用（N回目）]
  │
  ├── 乖離度 ≥ 8 ?
  │     │
  │     ├── YES → N += 1
  │     │         │
  │     │         ├── 主効果: 精神HP -= max(0, 1-mastery/2.0) × 0.04 × N × MentalMaxHP
  │     │         │
  │     │         ├── 乖離度 ≥ 12 ?
  │     │         │     ├── YES → 副次蓄積 += max(0, 1-mastery/3.0) × N
  │     │         │     │         ├── 蓄積 ≥ 6.0 → 精神DEF × 0.25
  │     │         │     │         ├── 蓄積 ≥ 3.0 → 精神DEF × 0.5
  │     │         │     │         └── 蓄積 < 3.0 → 精神DEF変化なし
  │     │         │     └── NO  → 副次蓄積は増えない
  │     │
  │     └── NO  → 消耗なし
  │
  ▼ 以降は既存システムが自動処理
[相手から攻撃を受ける]
  │
  ├── 精神ダメージ計算時、低下した精神DEFが適用
  ├── 精神HPがさらに削れる
  │
  ▼
[ターン終了時の乖離判定]  ── 総合戦闘仕様書
  │
  ├── 乖離率 = |精神HP - HP| / HP
  ├── 人間状況別しきい値と比較
  │
  ├── しきい値超過 + 持続ターン最大到達
  │     ├── TLOA → HP × 0.76 化
  │     └── 非TLOA → パッシブID 3（強制ダウナー）+ 50%でPower1Down
  │
  └── 乖離再充填: TentVoid × 3 − Miza ÷ 4 × Smiler
```

## 6. キャラ設計における活用例

> **注**: 本セクションは精神消耗システムを活かしたキャラ設計の**例示**であり、本仕様書のスコープ外。
> 具体的な実装を定義するものではない。

精神消耗は既存の仕組み（十日能力プロファイル、AI、ダウナー乖離パッシブ）の組み合わせで
キャラの個性を表現できる。例えば:

- **自滅型ボス**: 強力だが乖離したスキルを単純AIで連発する敵。放置すれば精神が自壊する
- **強化ダウナー**: `MentalDownerDiverGenceEffect()` をoverrideし、通常より重いデバフを持つキャラ
- **耐久戦略の価値**: 正面から殴り合わず、守りに徹して相手の精神崩壊を待つ、という戦略が生まれる

## 7. 実装詳細

### 7.1 処理タイミング

スキル使用開始時（`AttackChara` 末尾、行動記録の直後）に `ApplyMentalExhaustion()` を呼び出す。
ダメージ補正チェーンの外で動作する自傷処理。攻撃が当たったかどうかは問わない。

### 7.2 精神DEF低下の実装方式

パッシブではなく、`MentalDEF()` メソッドの戻り値に直接 `_mentalDefExhaustionMultiplier` を乗算する方式。

### 7.3 関連ファイル

| ファイル | 役割 |
|---------|------|
| `Assets/Script/Config/MentalExhaustionConfig.cs` | 全定数・計算メソッドを集約するstatic Config |
| `Assets/Script/BaseStates/Battle/BaseStates.BattleEvent.cs` | `GetIsSkillDivergence(out float)` + `ApplyMentalExhaustion()` |
| `Assets/Script/BaseStates/Battle/BaseStates.Attack.cs` | 呼び出し元（`AttackChara`末尾）+ `ActionSkillData` |
| `Assets/Script/BaseStates/BaseStates.StatesNumber.cs` | `MentalDEF()` に倍率乗算 |
| `Assets/Script/BaseStates/BaseStates.CallBack.cs` | `OnBattleStartNoArgument()` でリセット |

### 7.4 仮パラメータのシミュレーション（mastery=0, 最悪ケース）

| N | 今回消耗（%MaxHP） | 累計消耗 |
|---|---------|---------|
| 1 | 4% | 4% |
| 3 | 12% | 24% |
| 5 | 20% | 60% |
| 7 | 28% | 112%（枯渇） |

mastery=1.0 なら全て半分。mastery=2.0 で完全無消耗。

### 7.5 今後の検討事項

メモには「思えの値？」「精神攻撃率」の記述もある。
精神HP以外に消耗先を持たせるかは今後の検討事項。
